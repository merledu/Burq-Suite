# Burq Suite (v2.0)

## Setup
Clone the repo.
```shell
git clone --recurse-submodules https://github.com/merledu/Burq-Suite.git
cd Burq-Suite
```
On Arch Linux:
```shell
./arch_setup.sh
```
On Debian:
```shell
./debian_setup.sh
```

## Run
Execute `burq-suite.sh`
